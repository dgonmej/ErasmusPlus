Erasmus+ App
==========

Éste es el README de la aplicación ErasmusPlus.

Es una Aplicación web para buscar la localización de alumnos y profesorado de Eramus en Europa.

Herramientas utilizadas:
+ HTML 5
+ CSS 3
+ BootStrap
+ JavaScript
+ API de Google Maps
